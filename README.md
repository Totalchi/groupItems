# Shared Boss Uniques

Shared Boss Uniques is a RuneLite plugin for group-focused OSRS players who want to share boss collection-log unique progress.

## Features

- Captures boss collection-log pages as you browse them in-game
- Uploads scanned uniques to a shared sync server
- Shows combined group progress in a dedicated RuneLite panel
- Supports manual refreshes and manual uploads

## Usage

1. Start the companion sync server.
2. Create or join a private group and copy the group code.
3. Install this plugin in RuneLite.
4. Configure the sync server URL and group code in the plugin settings.
5. Open boss pages in the Collection Log to scan your uniques.

## Notes

- The plugin tracks boss collection-log uniques, not live bank contents.
- A boss is only tracked after its Collection Log page has been opened on that account.
- Group data is synced through the companion server.

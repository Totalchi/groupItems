OSRS Group Sharing

Maak privégroepen voor gedeelde boss uniques

Maak eerst een privé-groep aan. De server genereert dan een unieke group code die alleen jouw spelers invullen in de RuneLite-plugin. Open in-game de relevante bosspagina's in de collection log, waarna de plugin die uniques synchroniseert. Zo blijven groepen volledig gescheiden van elkaar.

Nieuwe groep aanmaken

Na het aanmaken kom je op de overzichtspagina terecht. Daar zie je de gegenereerde group code die je met je eigen spelers deelt.

Bestaande groep opzoeken

Gebruik

  1. Start deze Spring Boot server op poort 8080.
  2. Maak hierboven een groep aan en noteer de group code.
  3. Installeer de plugin-jar uit runelite-plugin/build/libs in RuneLite.
  4. Geef elke deelnemende speler dezelfde group code en server URL http://localhost:8080 of je eigen host.
  5. Open per speler bosspagina's in de collection log, bijvoorbeeld General Graardor of Demonic gorillas.
  6. Bekijk het groepsoverzicht op /groups/{jouw-code}.

API

POST /api/groups
Body:
{
  "groupName": "Team Thomas"
}
